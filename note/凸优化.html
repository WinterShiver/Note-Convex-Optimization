凸优化 CH 1 凸集和凸函数

凸集和凸函数

  • 凸集（凸组合连线）
    凸集的性质：交，加，仿射，水平集合是凸集
  • 凸函数（凸组合的函数值）
    典型的凸函数：指数，负对数，最小二乘，仿射，p范数
  • 凸函数的一阶判定
    • ∀x,x0,f(x)≥f(x0)+▽Tf(x0)(x−x0)\forall x,x_0,f(x)\geq f(x_0)+\bigtriangledown^Tf(x_0)(x-x_0)∀x,x0​,f(x)≥f(x0​)+▽Tf(x0​)(x−x0​)

    • ∀x,▽2f(x)\forall x,\bigtriangledown^2f(x)∀x,▽2f(x)对称半正定
      （正定：∀x,xTAx≥0\forall x,x^TAx\geq0∀x,xTAx≥0，则称矩阵AAA是半正定的）
    • 直观理解：二次函数二次项系数大于0
    • 一个实例：标准型二次函数f(x)=12xTpx+qTx+rf(x)=\frac{1}{2}x^Tpx+q^Tx+rf(x)=21​xTpx+qTx+r
      凹凸性：因为▽2f(x)=p\bigtriangledown^2f(x)=p▽2f(x)=p，所以判断ppp的正定性即可
  • 凸函数的性质（“xx也是凸函数”）：
    • 非负系数线性和
    • 仿射复合：f(Ax+b)f(Ax+b)f(Ax+b)
    • 凸集上的下确界（降维，注意形式）
    • 任意集合上的上确界（降维，注意形式）
    • 共轭必凸（造凸函数的方法）
    • 几个凸函数取max
    • Jensen：线性组合（系数和为1，保内部），引申到概率

无约束最小化方法：通用下降方法

求解无约束问题使用基于迭代的方法，取一点的序列以迫近梯度为0的点

find x # 取初值点
while |gradient x|>e # 有时也会采取不是梯度范数的其他停止条件
# 有时这里也用|gradient x|>2me，e对应原函数误差
   find d # 找下降方向d
   # 下降方向：梯度与方向内积<0，即局部一阶凸
   find t # 找下降步长t
   # 下降步长：只要使函数值下降即可，即f(x+td)<f(x)
   upgrade x: x=x+td
end
  • 以下记原始误差e0=e(x0)=f(x0)−p∗e_0=e(x_0)=f(x_0)-p^*e0​=e(x0​)=f(x0​)−p∗。下文讨论收敛性时，实际讨论的eee也是达到f(x)−p∗&lt;ef(x)-p^*&lt;ef(x)−p∗<e的情况,不是函数梯度模的eee。算法中用梯度表示迭代停止的停止阈值对应这里的2me2me2me。
  • 这里梯度没有要求具体是哪个范数，不过回溯下降和2范数联系紧密，这里最好采取2范数；最速下降最好采取和最速方向确定相同的范数。牛顿下降使用的是牛顿减小量。

确定下降方向的办法

  • 梯度下降：取负梯度方向为下降方向
  • 最速下降：基于方向导数，找下降速度最快的方向

  • 非规范化最速下降：最速下降方向*梯度的对偶范数
